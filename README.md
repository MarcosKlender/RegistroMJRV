# RegistroMJRV
 Web app for the tracking and registration of +8600 MJRV.

## Data for Excel Imports
Please make sure your Excel files have the following columns:

#### MJRV

- provincia
- canton
- parroquia
- zona
- recinto
- institucion
- junta
- sexo
- cedula
- nombre
- correo
- correo2
- celular

#### Users

- name
- email
- username
- phone
- location
